What is Universal File Encrypter?

The Universal File Encrypter is a straightforward tool for securing your important files. It uses strong encryption to protect your documents, images, and other data from unauthorized access.

Protect sensitive information with a password you set. The app features an easy-to-use graphical interface, built with Rust.

Key Features

🛡️

Strong Encryption

Powered by AES-GCM for robust security.

🔑

Password Protected

Only you (with the password) can access your encrypted files.

📄

PDF Disguise Option

Optionally embed encrypted data within a standard PDF file.

🖥️

Simple GUI

User-friendly graphical interface. No command line needed.

For macOS & Linux Users

For macOS and Linux, you can compile the application from its source code. This requires having Rust installed on your system.

Compilation Instructions:

  1. 1. Get the Source Code (Clone Repository):
    git clone https://github.com/RIZAmohammadkhan/file_encrypter.git
  2. 2. Enter the Project Directory:
    cd file_encrypter
  3. 3. Build the Application:
    cargo build --release
  4. 4. Locate the Executable:
    The compiled application will be in the `target/release/` directory. (e.g., `target/release/file_encrypter`)
